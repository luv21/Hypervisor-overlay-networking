Software for CRUD options on Subnets


vx_subnet script:

------------------

python vx_subnet.py

2. Customer is prompted for the password

3. Subnet id


delete_subnet script:

--------------------

python delete_subnet.py


1. Customer is asked for his Tenant id.

2. Which subnet customer wants to delete.

update_subnet:

-------------
